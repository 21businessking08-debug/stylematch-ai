# Архитектура и технологии проекта

## Какие технологии будем использовать

### Для внешнего вида (Frontend)
- React — для веб-версии.
- React Native — для мобильных приложений (iOS и Android).
- React Router — для навигации между страницами.

### Для серверной части (Backend)
- FastAPI (язык Python) — сервер, который будет обрабатывать запросы.
- PostgreSQL — основная база данных.
- Firebase — для хранения картинок и быстрых уведомлений.

### Для искусственного интеллекта
- Clarifai API — готовый сервис для распознавания одежды.
- OpenAI GPT — для умных описаний (если понадобится).

### Где будет работать программа (хостинг)
- Vercel — для веб-версии.
- Render — для серверной части.
- GitHub — здесь хранится весь код.

## Схема работы
Пользователь нажимает кнопку → запрос уходит на сервер → сервер спрашивает у AI → AI отвечает → сервер отдаёт результат обратно в приложение.
