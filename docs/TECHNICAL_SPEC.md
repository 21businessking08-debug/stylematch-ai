# Техническое задание на разработку StyleMatch AI

## 1. Цель проекта
Создать мобильное приложение, которое помогает пользователям подбирать образы на основе загруженной одежды, параметров тела и предпочтений в стиле.

## 2. Целевая аудитория
Молодые люди 16–35 лет, интересующиеся модой и персонализированными рекомендациями.

## 3. Функциональные требования

### 3.1. Модуль распознавания одежды
- Пользователь загружает фото вещи.
- Система определяет тип вещи (футболка, джинсы, обувь).
- Система определяет цвет.
- Система определяет стиль (кэжуал, спорт, деловой).

### 3.2. Модуль генерации образов
- На основе распознанной вещи система подбирает 3–5 готовых образов.
- Каждый образ состоит из нескольких предметов одежды.
- У каждого образа есть свой стиль.

### 3.3. Модуль аватара
- Пользователь настраивает 2D-аватар по параметрам: рост, вес, процент жира.
- Аватар визуально меняет пропорции.
- Можно "примерить" образ на аватаре.

### 3.4. Модуль разбора образа
- При клике на образ он раскладывается на отдельные вещи.
- Каждую вещь можно посмотреть отдельно.
- Можно сохранить образ в избранное.

## 4. Нефункциональные требования
- Время ответа от сервера не более 2 секунд.
- Приложение должно работать на iOS и Android.
- Данные пользователей должны быть защищены.
