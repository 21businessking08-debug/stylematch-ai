# Определение MVP (Minimum Viable Product)

Это список того, что будет сделано в первую очередь. Остальное — позже.

## Что входит в первую версию (MVP)
- [x] Загрузка фото из галереи телефона.
- [x] Распознавание типа одежды и цвета через готовый сервис.
- [x] Показ 3–5 образов (сначала это будут просто картинки-заглушки).
- [x] Простой 2D-аватар, который меняет размер от ползунков.
- [x] Возможность сохранить образ в избранное (на самом телефоне).
- [x] Веб-версия, чтобы можно было тестировать на телефоне через браузер.

## Что НЕ входит в первую версию (будет позже)
- Регистрация и личные кабинеты.
- Лайки, комментарии и другие социальные штуки.
- Покупки через интернет-магазины.
- Свои собственные нейросети (используем готовые).
- 3D-аватар.
