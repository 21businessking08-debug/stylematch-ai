# StyleMatch AI

Ваш персональный стилист на основе искусственного интеллекта.

Загрузите фото любой вещи из вашего гардероба, и StyleMatch AI подберёт несколько стильных образов, которые можно примерить на вашем персональном 2D-аватаре.

## Основные возможности

- 📸 Распознавание одежды по фото.
- 👕 Генерация готовых образов.
- 🧍‍♂️ Примерка на аватаре с вашими параметрами.
- ❤️ Сохранение понравившихся луков.

## Как начать

1. Скопируйте репозиторий к себе на компьютер.
2. Перейдите в папку frontend и установите зависимости командой npm install.
3. Запустите проект командой npm start.

Подробная документация — в папке [docs](docs/README.md).

## Технологии

- Frontend: React, React Native
- Backend: FastAPI
- AI: Clarifai, OpenAI
- База данных: PostgreSQL, Firebase
